flask实现的简单进销存后台管理系统
=======
## 进销存管理后台系统



> flask实现的进销存后台管理系统，包含个模块：采购管理，销售管理，仓库管理，数据统计，管理员管理，系统管理。

python manage.py runserver
#默认超级管理员
